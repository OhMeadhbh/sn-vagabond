This app could not have been made without some exceptional open source packages. It would be gauche (and a violation of some licenses) if we didn't alert the world to their awesomeness.

For rendering and displaying web pages, we use jQuery, Backbone.js, Bootstrap and Handlebars.js.

jQuery is an extremely popular tool for (among other things) manipulating elements on a web page and enabling cross-browser XHR support. It is released under a dual MIT / GPL license. jQuery uses the Sizzle selector engine internally which is released under MIT, BSD and GPL licenses.

Backbone.js is a MVC (Model-View-Controller) framework for JavaScript applications. Backbone is copyrighted by Jeremy Ashkenas and DocumentCloud but released under a MIT license. Backbone.js is mature, functional, well-documents and it's easy to find examples of its use.

Bootstrap forms the core of our CSS styling and many user elements. It is released under an Apache License and is awesome.

We use a templating library from Yehuda Katz called Handlebars.JS. Think of it as "Mustache on steroids," and you wouldn't be far off. Handlebars is released under an MIT license.

On the server side we use Node.js as a framework for our application logic. We use the popular Connect package to manage routing HTTP requests within our app.

This interface was made with larb, a collection of useful scripts, HTML fragments and node weirdness for configuring headless linux computers. It also is released with a MIT License, so feel free to use it!

We owe an enormous debt of gratitude to the individuals responsible for these packages. Track us down sometime, we'll buy you a beer.
